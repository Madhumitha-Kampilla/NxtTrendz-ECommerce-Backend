# NxtTrendz E-Commerce Backend

## 🚀 Project Overview
This is a backend application built using Spring Boot.  
It provides REST APIs for managing products and categories.

---

## 🛠 Tech Stack
- Java
- Spring Boot
- REST APIs
- Maven

---

## 📂 Project Structure
- controller/ → Handles API requests  
- service/ → Business logic  
- repository/ → Database operations  
- model/ → Data models  

---

## ⚙️ Features
- Create, Read, Update, Delete (CRUD) operations
- RESTful API design
- Layered architecture

---

## ▶️ How to Run
1. Clone the repository  
2. Open in IDE (IntelliJ / Eclipse)  
3. Run `NxtTrendzApplication.java`  
4. Test APIs using Postman
